TinyMCE Grid

Client: Exware AMS

Tools: HTML, CSS, TinyMCE

Description: A tool for inserting responsive grids into websites using the Exware system. I teamed up with a backend developer to implement this feature where clients can add and update board members, portfolios, and other grid content without having to code. It provides an alternative to tables which can present issues with responsive design and be difficult for non-technical users.
