Keep Footers at the Bottom

Footers will leave a blank space underneath if there isn't enough content to fill the height of the screen. A fix is to relatively position the page, give it a full height and use padding to ensure the footer stays at the bottom without overlapping content.

The Fix

Wrap the page in a div:

{% highlight html %}
{% endhighlight %}

Add the following CSS:

{% highlight css %} html, body { height:100%; } .page-wrap { min-height:100%; position:relative; } main { padding-bottom: 60px; } #footer { position:absolute; bottom:0; width:100%; height:60px; } {% endhighlight %}
